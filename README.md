### フィッシング詐欺防止3DCGアニメーション
**警察庁主催「サイバー防犯ボランティア広報啓発コンテスト」  
フィッシング対策部門 最優秀賞（サイバー警察局長賞）受賞**

- **概要：** フィッシング詐欺の仕組みを、子どもでも直感的に理解できる3DCGアニメーションとして制作
- **コンセプト：** 「釣り」に見立て、詐欺師＝猫／被害者＝犬として表現
- **内容：** 偽の再配達メールから個人情報を入力してしまう流れを40秒のストーリーで可視化
- **工夫：** 難解なサイバー犯罪を“ポップで理解できる体験”に変換
- **役割：** 企画・脚本・モデリング・アニメーション・編集を担当（約2ヶ月で制作）

### [360°3D映像のシームレス合成システム](https://github.com/yoshiototora/360-video-synthesis-v1)
**低コストで自然なVR映像を生成する映像処理システム**

- **課題：** 360度映像の境界で被写体が不自然に切断される「境界問題」
- **解決：** 深度推定AI + 動的閾値による前景分離 + 多段階フェード処理
- **結果：** 主観評価実験にて従来手法より自然さが有意に向上（p < 0.01）
- **技術：** Python / PyTorch / OpenCV

### [HabitSpark（iOSアプリ）](https://apps.apple.com/jp/app/habitspark-%E6%8C%AF%E3%82%8A%E8%BF%94%E3%82%8A%E3%81%A7%E7%B6%9A%E3%81%8F%E3%82%B7%E3%83%B3%E3%83%97%E3%83%AB%E7%BF%92%E6%85%A3%E5%8C%96%E3%82%A2%E3%83%97%E3%83%AA/id6762178668)
